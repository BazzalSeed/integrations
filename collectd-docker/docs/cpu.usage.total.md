### CPU Usage Total

Tracks how much CPU time, in jiffies, a container has used. To get a CPU utilization percentage, compute against [`cpu.usage.system`](cpu.usage.system.md):

```
CPU usage % = 100 * cpu.usage.total / cpu.usage.system
```
